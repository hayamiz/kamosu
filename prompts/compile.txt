Notes on reading source files:

- For large PDFs (over 10 pages), use the Read tool's `pages` parameter to read specific page ranges.
- The `pdftotext` and `pdfinfo` commands are available via Bash if needed for text extraction.
- Do NOT install packages or run any other Bash commands.
